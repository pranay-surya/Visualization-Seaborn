# Data Visualization Practice [ Seaborn ]

This repository contains practice files focused on understanding and implementing **data visualization using Seaborn**. It is built on top of matplotlib and integrates closely with pandas data structures to provide a high-level interface for creating attractive and informative visualizations.


---

## Concepts Covered

- Seaborn Basics & Styling
- Line plot
- Multi-line Plot
- Linear Model plot
- Heatmap
- Scatter plot
- Bar plot
- Histogram plot 
  

---

## Tech Stack

- Seaborn  
- Matplotlib  
- Python  
- NumPy  
- Pandas
